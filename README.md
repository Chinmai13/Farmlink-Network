# FarmLink Network 🌾

FarmLink Network is a web application that connects farmers directly with buyers.
Built using HTML, CSS, JavaScript, and Firebase.

## Features
- Firebase Authentication
- Role-based access (Farmer / Buyer)
- Crop listing & browsing
- Feedback & reviews
- Multilingual support

## Tech Stack
- HTML, CSS, JavaScript
- Firebase Auth & Firestore
- Firebase Hosting
